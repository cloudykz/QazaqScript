# QazaqScript

**Қазақ тілінде бағдарламалауды үйрену платформасы.**

Кодты ойлағаныңыздай — қазақша — жазыңыз. QazaqScript оны жүйелік кодқа аударып, браузерде бірден орындайды.

> Ешбір бағдарлама орнату керек емес. Тек браузер.

---

## Кімге арналған?

| Сіз кімсіз? | Не істей аласыз? |
|-------------|------------------|
| **Толығымен жаңадан бастаған** | 10 қадамдық сабақтан өтіңіз |
| **Біраз код жазған** | Жазба алаңында эксперимент жасаңыз |
| **Мұғалім / оқытушы** | Дайын мысалдар мен сабақтарды пайдаланыңыз |
| **Әзірлеуші** | `lib/qazaqscript.js` кітапханасын өз жобаңызға қосыңыз |

## 3 қадамда бастау

### 1-қадам: Сайтты ашыңыз

**Ең оңай жол** — `index.html` файлын екі рет басыңыз (браузерде ашылады).

**Ұсынылатын жол** — кіші сервер іске қосыңыз (бөлісу және кейбір функциялар дұрыс жұмыс істейді):

```bash
python -m http.server 8000
```

Содан кейін браузерде ашыңыз: `http://localhost:8000`

> **Windows:** PowerShell-де алдымен папкаға өтіңіз:
> `cd C:\жол\QazaqScript`
> содан кейін жоғарыдағы команданы жазыңыз.

### 2-қадам: Жазба алаңын сынаңыз

1. **Жазба алаңын ашу** батырмасын басыңыз
2. Кодты өзгертіңіз
3. **Іске қосу** батырмасын басыңыз
4. Оң жақта аударма коды мен шығысты көріңіз

### 3-қадам: Сабақтан бастаңыз

**Сабақтар** бөліміне өтіп, 1-сабақтан бастаңыз. Әр қадамда тапсырма беріледі — **Тексеру** батырмасы дұрыс орындалғанын айтады.

## Бірінші бағдарлама (қадамдық)

`playground.html` ашып, мына кодты жазыңыз:

```text
"Сәлем, QazaqScript!" жаз.

аты "Айгүл" болсын.
аты жаз.
```

**Іске қосу** басыңыз. Шығыста екі жол болуы керек:

```text
Сәлем, QazaqScript!
Айгүл
```

Қате шықса — әр сөйлем `.` нүктемен аяқталғанын тексеріңіз.

## Өз сайтыңызға қосу

`lib/qazaqscript.js` файлын HTML-ге қосыңыз:

```html
<script src="lib/qazaqscript.js"></script>
<script>
  var нәтиже = QazaqScript.compile('"Сәлем!" жаз.');
  if (нәтиже.errors.length === 0) {
    // нәтиже.js — аударылған жүйелік код
    eval(нәтиже.js);
  }
</script>
```

`compile()` қайтарады:

- `js` — дайын жүйелік код
- `errors` — қателер тізімі (жол нөмірімен)

## Мысал немесе сабақ қосу

`assets/content.js` файлын ашыңыз.

**Жаңа мысал** — `EXAMPLES` массивіне қосыңыз:

```javascript
{
  "id": "menin-mysalym",
  "title": "Менің мысалым",
  "description": "Қысқаша сипаттама",
  "category": "негіздер",
  "code": "\"Сәлем\" жаз.\n"
}
```

**Жаңа сабақ** — `LESSONS` массивіне қосыңыз. Қазіргі сабақтарды үлгі ретінде қараңыз.

Сайтты жаңартып, браузерде тексеріңіз.

## Жиі қойылатын сұрақтар

**Бағдарлама орнату керек пе?**  
Жоқ. Бұл жоба тек веб файлдарынан тұрады. Ешнәрсе орнатпай-ақ жұмыс істейді.

**Интернет керек пе?**  
Сыртқы стильдер мен шрифттер интернет арқылы жүктеледі. Толық офлайн жұмыс істеу үшін интернет қажет.

**Файлды тікелей ашқанда не болады?**  
Сайт ашылады, бірақ бөлісу функциясы шектелуі мүмкін. `python -m http.server` пайдаланыңыз.

**Қате шықса не істеймін?**  
1. Әр сөйлем `.` нүктемен аяқталғанын тексеріңіз  
2. Жақшалар жабылғанын тексеріңіз  
3. **Сабақтар** бөлімінен өтіп, синтаксисті қадамдық үйреніңіз

**Қате хабарламалар қай тілде?**  
Қазақша.
